# RubyASCII
A simple collection of ASCII art to spice up your terminal sessions.

### Requirements
- ruby

### Installation
```bash
git clone https://github.com/Blziko/rubyascii
chmod +x rubyascii/rubyascii.rb
sudo ln -sf $HOME/rubyascii/rubyascii.rb /usr/bin/rubyascii
```

### Usage
```bash
rubyascii [--ascii=<number>]
```
